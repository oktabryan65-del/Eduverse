## Deskripsi Karya
#Deskripsi Karya
Eduverse adalah platform pembelajaran teknologi berbasis web yang menggabungkan sistem e-learning dengan gamifikasi, berjalan sepenuhnya di sisi klien dan menyimpan data pengguna di LocalStorage browser. Pengguna dapat mendaftar dan login secara mandiri dengan bonus awal 100 koin dan XP, serta memanfaatkan toko virtual untuk membeli booster atau kosmetik melalui sistem keranjang dan checkout. Setiap aktivitas seperti pembelian atau memulai kursus akan otomatis menambah XP dan meningkatkan level pengguna, sehingga menciptakan pengalaman belajar yang interaktif, seru, dan terstruktur.
